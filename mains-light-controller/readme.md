# Ceiling Light Controller

This component fits the general ceiling light socket in Denmark.



<hr>
<h3>Inputs</h3>
<ul>
    <li>M1: Power from switches</li>
    <li>M2: Constant power supply</li>
    <li>N: Neutral</li>
</ul>
